This folder contains simple scaffolded layout components:

- `Layout.jsx` - top-level app shell that arranges header, menu, sidebar, content, and footer.
- `Header.jsx` - simple header component
- `Menu.jsx` - top navigation/menu
- `Sidebar.jsx` - side navigation
- `Content.jsx` - content wrapper
- `Footer.jsx` - footer component

Use `Layout` by passing components as props and placing page content as children.
